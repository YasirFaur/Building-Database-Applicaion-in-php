Course Overview
Welcome to Building Database Applications in PHP! In this course, you will learn how to use PHP and MySQL to make database applications. You will study PDO, SQL commands, cookies, and sessions. You will learn how to log in users and show messages in your web applications. At the end, you will build a complete application to Create, Read, Update, and Delete data.
