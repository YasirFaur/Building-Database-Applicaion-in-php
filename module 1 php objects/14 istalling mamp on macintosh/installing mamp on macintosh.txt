In this video, we install MAMP on a Mac. First, we download MAMP from MAMP.info and run the installer. We keep the default settings. After installing, MAMP is in Applications. We start it from the MAMP control panel. We check phpinfo() to see the PHP version and settings. For development, we want to see errors, so we open the php.ini file in Applications → MAMP → bin → php → php 7.1.0 → conf. We turn on display_errors and display_startup_errors. Then we save the file and restart MAMP.

Next, we make a simple web page. We create a folder called First in MAMP → htdocs. We save a file called index.php with the text “Hello, from my first web page.” We open localhost:8888/First in the browser. We can see our page. Now MAMP is working, errors are showing, and our first PHP page runs correctly.
