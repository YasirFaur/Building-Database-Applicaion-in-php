رحلتي من الإلكترونيات إلى تطوير مشاريع IoT عبر الويب

لم أتعلم HTML وCSS وJavaScript وPHP لأصبح مطور ويب تقليدي، بل لإنجاز مشاريع الأنظمة المدمجة والإنترنت الأشياء (IoT).

البداية كانت مع ESP32 لإنشاء نظام تحكم ومراقبة عبر الهاتف، حيث كان يجب أن أطور واجهة مستخدم تظهر على جهاز العميل عند الاتصال بالمشروع عبر Wi-Fi. هذه كانت المرة الأولى التي أدمج فيها HTML وCSS وJS مع لغة C، لإرسال واستقبال البيانات وعرض القيم والأوامر على صفحة ويب مباشرة.

لتعلم هذه اللغات، اعتمدت على مصادر عربية وإنجليزية، وأفضلها على منصة كورسيرا، حيث أنهيت أول موقع شخصي لي يدوياً، سطرًا بسطر:
https://yasirfaur.github.io/EmbeddedSystemsWebsite/

مشروعي على كورسيرا كان ضمن تخصص Web Design for Everybody: Basics of Web Development & Coding، وشمل خمس دورات تدريبية. لاحقًا، تعلم PHP سمح لي بإدارة قواعد البيانات على الخادم وفهم أعمق لكيفية عمل ESP32 كخادم أو عميل أو نقطة وصول، مع مراعاة الأمان والتفاعل الديناميكي مع المستخدم.

أنا من الجيل الذي تعلم الإلكترونيات التقليدية، ولم أتخيل أنني سأكتب أكواد ويب يومًا، لكن دمج الإلكترونيات بالويب أتاح لي بناء مشاريع IoT تتفاعل مباشرة مع المستخدم، وتتجاوز قيود الزمان والمكان ضمن نفس الشبكة.

البداية كانت مع مشروع اعتمدت فيه على ESP32 للتحكم والمراقبة عبر الهاتف الجوال. كان يجب أن أُنشئ واجهة مستخدم تظهر على جهاز العميل عند اتصاله بالمشروع عبر Wi-Fi، حيث كان ESP32 يعمل كنقطة وصول تظهر كشبكة واي فاي على الهاتف.
